# Mini ERP System

A lightweight ERP-style system for managing inventory, sales, and business insights.

## Features
- Add and manage products
- Track inventory
- Process sales
- Dashboard with revenue insights
- Top-selling product analytics

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS/JavaScript

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install flask flask-cors
3. Run backend:
   python app.py
4. Open frontend using Live Server
